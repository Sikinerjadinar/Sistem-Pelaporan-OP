Panduan Setup Lengkap

Integrasi Apps Script + Google Drive (dengan foto)

ℹ️ Sistem ini menggunakan Google Apps Script Web App — bukan Google Forms biasa — sehingga foto bisa dikirim langsung ke Google Drive.
1

Buat Google Spreadsheet

Buka sheets.google.com → buat spreadsheet baru → beri nama "SiPOP – Data Laporan". Salin Spreadsheet ID dari URL:

https://docs.google.com/spreadsheets/d/SALIN_ID_INI/edit
2

Buat Apps Script

Di spreadsheet → klik menu Ekstensi → Apps Script → hapus semua kode lama → paste kode Apps Script. Ganti baris:

const SPREADSHEET_ID = "ID_SPREADSHEET_ANDA";

Ganti ID_SPREADSHEET_ANDA dengan ID yang disalin di langkah 1.

3

Deploy sebagai Web App

Di Apps Script → klik tombol Deploy → New Deployment → pilih tipe "Web App" → atur:

Execute as : Me Who has access: Anyone

Klik Deploy → izinkan akses → salin Web App URL yang muncul.

4

Pasang URL ke file HTML

Buka file HTML ini di editor teks → cari bagian KONFIGURASI → ganti URL:

const APPS_SCRIPT_URL = "PASTE_WEB_APP_URL_DISINI";
⚠️ Setiap kali kode Apps Script diubah, deploy ulang dengan klik Deploy → Manage Deployments → Edit → New Version.
5

Deploy HTML ke Netlify / GitHub Pages

Daftar di netlify.com → klik Add new site → Deploy manually → drag & drop folder berisi index.html → link siap dalam hitungan detik!

https://nama-anda.netlify.app/
6

Uji coba & bagikan ke tim

Isi form sekali → cek spreadsheet → pastikan data dan link foto masuk. Bagikan link ke petugas lewat WhatsApp. Sarankan untuk Add to Home Screen di HP masing-masing.
